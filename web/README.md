### main configuration of the project ：
- react 18+
- webpack 5+
- react-router-dom 6
- react-router
- antd
- redux
- react-redux

### install
- yarn 
- or
- npm run install

### start command：
- npm run start:open
- or
- yarn start

### build command:
- npm run build:open
- or
- yarn build:open

### support development：
- *.css
- *.less
- js/jsx
- ts/tsx
